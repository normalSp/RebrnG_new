# RebrnG Sprint 7 第一卷剧情骨架与长正文叙事试制

Status: frozen candidate  
Date: 2026-04-28  
Scope: Sprint 7

Sprint 7.1 refinement: 第一卷详细剧情生产以 `2026-04-28-reborng-first-volume-story-bible.md` 和 `first-volume-story-beats.json` 为准；本文件保留为 Sprint 7 开篇长正文试制范围说明。

## Summary

Sprint 7 的目标不是继续横向扩机制，而是先把《蛊真人》第一卷在游戏里的呈现母骨架定下来。当前项目已经有 8 自由窗口、月光蛊状态、四路线文本和 DeepSeek 离线候选工具，但还缺少“第一卷如何分章推进、玩家能影响什么、原著硬事实不能改什么”的统一剧情合同。

本 Sprint 采用“第一卷全局骨架 + 开篇长正文试制”的方式推进：

- 第一卷终点默认定义为“青茅山变局后，玩家获得脱离青茅山的资格或被迫离开原秩序”。
- `canon_strict` 与 `sandbox_if` 共用同一剧情骨架，但模式门禁不同。
- `canon_strict` 保护原著硬事实，玩家主要承受余波、经营缝隙、争取生存资格。
- `sandbox_if` 允许有限偏移，但必须有代价、阶段门槛和锚点压力，不给无代价超模收益。
- Sprint 7 只把“开窍大典到学堂初压”的长正文试制入库；第一卷后续段落只落文档和数据表。

内容版本策略：若开篇试制文本入库，`ContentManifest.version` 升为 `s0.6.0`。规则版本保持 `sprint4-rules-v1`，除非后续修改 AP、资源、遭遇、蛊虫或存档结算。

## Narrative Product Target

最终体验应接近“人生重开叙事流 + 冷峻账本策略”的混合体，而不是纯聊天或纯表格：

- 强锚点给足正文密度：3-6 段处境文本、4 个强叙事选择、2-5 段结果反馈。
- 普通行动保持短反馈：说明处境、代价、因果和下一步压力，不把每个按钮都写成大段小说。
- 玩家每次点击后都应知道“我做了什么、失去了什么、换来了什么、谁因此看见了我”。
- 文本不得替代规则事实；奖励、伤势、资源、蛊虫状态仍由 Rust 规则层写入。
- 账本明细必须显示定量变化；对话流负责解释定量变化的处境意义。

## First Volume Structure

### 1. 开窍大典与入学

玩家处境：空窍已开，资质与出身被家族秩序记账，玩家从“普通凡人”进入“可被培养、也可被压榨”的学堂秩序。

`canon_strict`：

- 开窍是制度入口，不是爽文抽奖。
- 玩家可以影响初始资源包、关注度、路线倾向和压力来源。
- 玩家不能跳过开窍、不能未开窍修行、不能直接获得稳定本命蛊或完整传承。

`sandbox_if`：

- 可允许更强天赋、异常征兆或残缺记忆，但只作为高代价 IF 钩子。
- 强 IF 不得直接改写原著主线硬事实。

机制需求：

- 已有设置层、属性、开窍结果、初始资源包。
- 后续需要更细的资质/空窍容量与冲刷进度。

### 2. 月光蛊根基

玩家处境：月光蛊从学堂制度内渠道进入玩家账本，领取、炼化、修行和喂养压力构成第一条真实蛊虫闭环。

`canon_strict`：

- 月光蛊是首发唯一稳定 canon 蛊虫闭环。
- 修行必须依赖已炼化月光蛊，且消耗窗口和元石。
- 玩家不能把月光蛊写成普通装备，也不能无成本跳过炼化。

`sandbox_if`：

- 可出现月光蛊异常反应、残线提示或偏门用法，但不能直接形成完整杀招库。

机制需求：

- 已有 `GuSpec / GuInstance / RefinementState / FeedingState`。
- 后续需要空窍冲刷、小境界推进、蛊虫损伤与喂养周期。

### 3. 学堂压力与家族秩序

玩家处境：修行痕迹被看见后，公开比较、对练压力、名声账和组织审计开始压缩自由窗口。

`canon_strict`：

- 玩家可以忍让、争辩、硬顶或绕行，但每种选择都有后果。
- 学堂压力是制度压迫，不是简单打脸爽点。
- 玩家不能稳定压制学堂秩序，也不能硬绑定原著核心人物替自己兜底。

`sandbox_if`：

- 可出现局部反制、嫁祸、假线索或短期声望偏移，但必须增加暴露或债务。

机制需求：

- 已有公开压力遭遇、决断、线索缓解。
- 后续需要更完整的名声、同辈关系和组织惩罚。

### 4. 酒虫与花酒传闻残线

玩家处境：酒虫、花酒行者传承和山寨暗线应作为第一卷重要诱惑，但在 S0 只能以传闻、残线、假线索和高风险查验存在。

`canon_strict`：

- 不稳定发放酒虫。
- 不让玩家完整获得花酒行者传承。
- 不允许提前抢走原著核心机缘或硬改写方源相关事实。

`sandbox_if`：

- 可开放残缺传闻、假入口、局部偏移和高代价试探。
- 即使偏移，也必须保留锚点压力、误导风险和失手代价。

机制需求：

- 需要传承残线系统、真假线索、证据等级和追索后果。
- 需要酒虫作为后续特殊蛊虫内容，但不在 S0 稳定获得。

### 5. 山野与狼潮前兆

玩家处境：玩家逐步从学堂和山寨缝隙走向山野，安全区更小，移动、侦查、恢复和撤退都要更重。

`canon_strict`：

- 狼潮前兆只能逐步显现，不能提前让玩家完全预知或稳定规避。
- 山野不是刷怪区，而是资源、视野和死亡风险交叠的空间。

`sandbox_if`：

- 可允许玩家更早接触异常风声，但必须换来暴露、伤势或路线断裂。

机制需求：

- 需要更大节点图、山野风险表、长跳时和受伤恢复节奏。
- 需要更多遭遇模板和撤退代价。

### 6. 三寨与白凝冰压力

玩家处境：三寨格局和白凝冰压力代表第一卷中后段的外部强压，玩家不应轻易同层竞争，只能在余波和缝隙中求活。

`canon_strict`：

- 白凝冰等原著强者不能被玩家早期稳定击败、控制或绑定。
- 玩家可以承受余波、借势避险、获取风声或被迫迁移。

`sandbox_if`：

- 可有高代价短接触、误判、被利用或局部 IF，但不能抹掉原著角色强度。

机制需求：

- 需要跨节点事件压力、势力关系和高阶 NPC guarded 模型。
- 需要更成熟的锚点偏移与主线保护。

### 7. 青茅山变局与脱离资格

玩家处境：第一卷阶段终点不是爽快通关，而是玩家在变局后获得离开青茅山秩序的资格、理由或被迫流亡条件。

`canon_strict`：

- 青茅山变局作为第一卷硬锚点必须被保护。
- 玩家不能无代价改写大结局，也不能让核心人物命运被早期选择轻易翻盘。
- 阶段成功是“活下来并带着可解释资产/关系/伤痕离开”，不是赢下青茅山。

`sandbox_if`：

- 可允许局部人物、资源和路线偏移，但必须写清楚世界反噬、锚点压力和后续追索。

机制需求：

- 需要阶段结算、跨地图迁移、资产容器转移和长期关系债。
- 需要脱离青茅山后的世界空间接口。

## Sprint 7 Implementation Scope

本 Sprint 只入库以下开篇试制内容：

- 开窍大典余波与学堂入账。
- 学堂门前开局场景长正文。
- 月光蛊领取、炼化、修行余波的长反馈。
- 方源邻近风声，只表现为危险邻近与余波，不绑定、不合作、不控制。
- 第一轮学堂公开压力的触发和三种决断反馈。

不在 Sprint 7 实现：

- 完整第一卷可玩内容。
- 空窍冲刷、小境界推进、蛊虫升转、完整炼蛊、杀招雏形。
- 酒虫稳定获得、花酒行者完整传承、方源硬绑定、提前改写青茅山结局。

## Canon And IF Redlines

`canon_strict` 禁止：

- 稳定获得酒虫或完整花酒行者传承。
- 方源硬绑定合作、被玩家控制、被提前击败或命运被早期选择硬改。
- 稳定本命蛊建立。
- 无代价越阶收益、跳过开窍、跳过炼化、跳过元石/AP 成本。
- 把黑市写成宝黄天前身，或把传承写成无风险宝箱。

`sandbox_if` 允许但必须门禁：

- 强天赋、异常梦兆、残缺记忆、偏移传闻。
- 高风险传承试探、假线索和局部偏移。
- 以伤势、债务、暴露、锚点压力、关系断裂作为代价的非稳定收益。

## Acceptance

- 第一卷 7 段骨架均明确 `canon_strict / sandbox_if` 差异。
- 后续机制缺口被登记，而不是在剧情文本里临时发明。
- 入库文本只覆盖开篇试制范围，且不改变规则结算。
- 玩家走查时能明确感到“开窍大典是主线入口”，而不是背景设定。
- 点击前 3-5 个关键行动后，对话流文本量与因果反馈明显比 Sprint 6 更厚。
